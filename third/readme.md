
# Setup

你需要在此处创建文件夹`lib`，从r200中提取以下库文件放入文件夹`lib`中

项目使用的固件版本是`6.00.7`和`6.00.8`，不同版本可能需要重新适配工具链版本和`libcurl`版本

```
libcurl.so        libgmp.so.10.3.2  libgnutls.so.30.22.0  libidn2.so.0      libmeigapps.so  libnettle.so.6.4       libz.so.1
libcurl.so.4.5.0  libgnutls.so      libhogweed.so.4       libidn2.so.0.3.4  libmeigbase.so  libunistring.so.2      libz.so.1.2.11
libgmp.so.10      libgnutls.so.30   libhogweed.so.4.4     libmal.so         libnettle.so.6  libunistring.so.2.1.0
```
